# Go To-Do List (CLI Application)

A simple, persistent **Command-Line To-Do List Application** built in **Golang**.  
This project helps beginners learn practical Go concepts such as:

- Structs & slices
- JSON encoding/decoding
- File handling (read/write)
- CLI menus
- CRUD operations
- Persistent storage
- Program flow control
